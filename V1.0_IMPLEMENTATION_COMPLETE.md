## 🗄️ Database Schema Changes

### New Tables

#### 1. `conversation_sessions`
Tracks multi-turn conversation sessions.

| Column | Type | Description |
|--------|------|-------------|
| Id | GUID | Primary key |
| UserId | VARCHAR(100) | User who owns session |
| Title | VARCHAR(500) | Auto-generated title |
| CreatedAt | DATETIME | Creation timestamp |
| LastInteractionAt | DATETIME | Last activity timestamp |
| IsActive | BOOL | Active status |
| MessageCount | INT | Total messages |
| Metadata | VARCHAR(4000) | JSON metadata |

**Indexes:**
- `UserId`
- `UserId + IsActive`
- `LastInteractionAt`

#### 2. `conversation_messages`
Individual messages within sessions (Q&A pairs).

| Column | Type | Description |
|--------|------|-------------|
| Id | GUID | Primary key |
| SessionId | GUID | Foreign key to sessions |
| UserId | VARCHAR(100) | User ID |
| Role | VARCHAR(20) | "user" or "assistant" |
| Content | TEXT | Message content |
| ReferenceChunks | VARCHAR(4000) | JSON array of chunk IDs |
| SequenceNumber | INT | Order in session |
| CreatedAt | DATETIME | Message timestamp |
| CostSeconds | DECIMAL(18,6) | Processing time |
| IsSuccess | BOOL | Success flag |
| SearchSimilarity | FLOAT | Avg similarity score |
| ContextTokenCount | INT | Context tokens |
| PromptTokenCount | INT | Prompt tokens |
| UsedHyDE | BOOL | Was HyDE used? |
| UsedMultiQuery | BOOL | Was Multi-Query used? |
| UsedSelfReflection | BOOL | Was Self-Reflection used? |
| SelfReflectionConfidence | INT | Confidence score (0-100) |

**Indexes:**
- `SessionId`
- `SessionId + SequenceNumber`
- `CreatedAt`

**Relationships:**
- `SessionId` → `conversation_sessions.Id` (CASCADE DELETE)

## 🔧 Service Layer

### New Services

#### 1. `IConversationMemoryService`
Manages conversation history and context.

**Key Methods:**
- `CreateSessionAsync()` - Create new conversation
- `GetRecentHistoryAsync()` - Get recent messages (token-limited)
- `BuildContextAwarePromptAsync()` - Prompt with history + citations
- `AddMessageAsync()` - Save user/assistant messages
- `ArchiveInactiveSessionsAsync()` - Cleanup old sessions

#### 2. `IHybridSearchService`
Combines vector and keyword search.

**Key Methods:**
- `SearchAsync()` - Hybrid search with RRF fusion

**BM25 Implementation:**
- Simplified Okapi BM25 algorithm
- Chinese + English tokenization
- SQL-based candidate filtering (can upgrade to Lucene.NET)
- Reciprocal Rank Fusion with vector results

#### 3. `IQueryRewritingService` (Enhanced)
Query understanding and expansion.

**Methods:**
- `GenerateHypotheticalDocumentAsync()` - HyDE technique
- `GenerateSimilarQueriesAsync()` - Multi-query generation
- `DecomposeQueryAsync()` - Complex query breakdown

#### 4. `ISelfReflectionService` (Enhanced)
Answer validation and correction.

**Methods:**
- `ValidateAnswerAsync()` - Check answer against sources
- `SelfCorrectAsync()` - Auto-improve low-confidence answers

## 🚨 Troubleshooting

### Memory System Not Working

**Symptom:** No conversation history in follow-up questions.

**Check:**
1. Migration applied? `SELECT * FROM conversation_sessions;`
2. Logs show: `💭 Using conversation memory`?
3. If not, check DI registration in Program.cs

**Fix:**
```bash
dotnet ef database update
```

### Hybrid Search Always Falling Back

**Symptom:** Logs show: `Hybrid search failed, falling back to vector search`

**Check:**
1. `AppEnterpriseAiContext` injected in `HybridSearchService`?
2. `DocumentChunks` table has data?

**Fix:**
```csharp
// In Program.cs, ensure:
builder.Services.AddScoped<IHybridSearchService, HybridSearchService>();
```

### BM25 Returns No Results

**Symptom:** Logs show: `📊 BM25 results: 0`

**Possible Causes:**
1. Query has no valid terms (all punctuation/short words)
2. No documents match permission filters
3. Database has no chunks

**Check:**
```sql
SELECT COUNT(*) FROM document_chunks;
SELECT * FROM document_chunks LIMIT 5;
```

### Self-Reflection Always Skipping

**Symptom:** No self-reflection logs.

**Check:**
1. Is `ISelfReflectionService` injected in `ChatUseCase` constructor?
2. Logs show: `Self-reflection failed, using original answer`?

**Fix:**
Check Ollama is running:
```bash
ollama list
curl http://localhost:11434/api/tags
```
