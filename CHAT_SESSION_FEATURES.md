## 🔧 技术实现细节

### 后端 API (ChatSessionController)

#### 获取会话列表
```http
GET /api/chat/sessions?limit=20
Authorization: Bearer {token}
```
- 从 JWT Token 自动获取用户 ID（更安全）
- 返回用户的活跃会话列表
- 按最后交互时间降序排列

#### 获取会话消息
```http
GET /api/chat/sessions/{sessionId}/messages
Authorization: Bearer {token}
```
- 返回会话详情和所有消息历史
- 消息按创建时间升序排列

#### 更新会话标题
```http
PATCH /api/chat/sessions/{sessionId}
Authorization: Bearer {token}
Content-Type: application/json

{
  "title": "新标题"
}
```

#### 删除会话
```http
DELETE /api/chat/sessions/{sessionId}
Authorization: Bearer {token}
```
- 软删除（IsActive = false）

#### 创建会话
```http
POST /api/chat/sessions
Authorization: Bearer {token}
Content-Type: application/json

{
  "title": "可选标题"
}
```
- 从 JWT Token 自动获取用户 ID

## 🎨 前端实现

### React Hooks

#### `useSessions()`
- 获取会话列表
- 自动启用（需要用户登录）
- 查询键：`['sessions']`

#### `useSessionMessages(sessionId)`
- 加载指定会话的消息
- 自动更新 chatStore 中的 messages
- 仅在 sessionId 存在时启用

#### `useUpdateSessionTitle()`
- 更新会话标题的 mutation
- 成功后自动刷新会话列表
- 显示成功/失败提示

#### `useDeleteSession()`
- 删除会话的 mutation
- 成功后自动刷新会话列表
- 如果删除当前会话，清空消息
